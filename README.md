# CHLE

Optimizing [LAMMPS](https://lammps.org) for the Deucalion Supercomputer (Large Scale Hybrid
Computing 2026 assignment). Our work achieved the following results:

 - **10.8% throughput improvement** over the default configuration;
 - **Checkpointing without performance penalty**, provided sharding is used;
 - Performance gains of up to 77.4% when using Infiniband instead of Ethernet;
 - Additional performance optimization strategies identified via profiling.

For more details, see [our report](TP2/report/report.pdf). Grade: 18.6 / 20.
